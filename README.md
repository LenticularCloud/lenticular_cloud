Lenticular Cloud
================


Simple user Manager in LDAP



Features
--------

* Web Platform to mange users
* OpenId / OAuth
* client certs
* ldap backend, can be used by other services

Auth Methods:
-------------
 * U2F
 * TOTP
 * Password
 * WebAuth (TODO)
